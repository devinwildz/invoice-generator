# 🚀 Invoice Generator

A modern, fully responsive **Invoice Generator Web App** built with Next.js, designed to create, manage, and download professional invoices with ease.

---

## ✨ Features

* 🧾 Create and customize invoices in real-time
* 📥 Download invoices as PDF
* 🔐 Secure authentication using Supabase
* ✅ Form validation with Zod
* 🎨 Clean and modern UI using Shadcn UI
* ⚡ Fully client-side optimized for fast performance
* 📱 Responsive design (mobile + desktop)

---

## 🛠️ Tech Stack

* ⚛️ Next.js
* 🎨 Shadcn UI
* 🗄️ Supabase
* 🧩 Zod

---

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/devinwildz/invoice-generator.git
cd invoice-generator
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Setup environment variables

Create a `.env.local` file and add:

```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```

---

### 4️⃣ Run the development server

```bash
npm run dev
```

Open 👉 http://localhost:3000 in your browser.
